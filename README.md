Business Task: Analyze smart device data to identify trends and provide marketing recommendations for Bellabeat.

Data Source: Fitbit dataset from Kaggle (30 users, ~1 month of data). Small sample size and limited time range may affect generalizability.

Tools Used: BigQuery (SQL), Google Sheets (data cleaning)

Data Cleaning: 
Fixed date formats
Removed duplicates
Created clean tables
Joined activity + sleep

Analysis & Insights: 
Positive correlation between steps and calories
Activity peaks on Saturday, drops on Sunday
Sedentary behavior highest on Thursday
Moderate users = largest segment
Activity intensity matters more than step count
Higher activity does not necessarily mean more sleep

Recommendations: 
Target moderate users to increase engagement
Introduce weekend activity challenges
Send Thursday motivation reminders
Promote intensity-based workouts
Encourage balanced activity and recovery
